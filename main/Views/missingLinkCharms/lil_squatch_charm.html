Lil' Squatch Charm Pattern Guide



Lil' Squatch Charm

The Lil' Squatch Charm is a remarkable collectible from the Missing Link Charm collection, which was added to the game on the 4th October 2024.


Patterns

Like any other charm in the game, the Lil' Squatch charm is pattern based. The pattern determines the look of the charm itself. For the Lil' Squatch charm it changes the color of the charm's headband, shorts, and socks, depending on the pattern it can be orange, yellow, green, blue, purple, or pink.


Orange Patterns

The low patterns on the charm will make the charm's clothes orange. As the pattern gets closer to 1 it will be more orange. This means pattern 1 is the best pattern and the pattern where the charm is as orange as it can get.

Yellow Patterns

The semi-low patterns on the charm will make the charm's clothes yellow. As the pattern gets closer to 10.000 it will be more yellow. This means pattern 10.000 is the best pattern and the pattern where the charm is as yellow as it can get.

Green Patterns

The mid-low patterns on the charm will make the char's clothes green. As the pattern gets closer to 30.000 it will be more green. This means pattern 30.000 is the best pattern and the pattern where the charm is as green as it can get.

Blue Patterns

The mid patterns on the charm will make the charm's clothes blue. As the pattern gets closer to 50.000 it will be more blue. This means pattern 50.000 is the best pattern and the pattern where the charm is as blue as it can get.

Purple Patterns

The high-mid patterns on the charm will make the charm's clothes purple. As the pattern gets closer to 80.000 it will be more purple. This means pattern 80.000 is the best pattern and the pattern where the charm is as purple as it can get.

Pink Patterns

The high-mid patterns on the charm will make the charm's clothes pink. As the pattern gets closer to 100.000 it will be more pink. This means pattern 100.000 is the best pattern and the pattern where the charm is as pink as it can get.
