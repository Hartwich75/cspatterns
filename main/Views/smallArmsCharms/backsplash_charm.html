Backsplash Charm Pattern Guide



Backsplash Charm

The Backsplash Charm is a high grade collectible from the Small Arms Charm collection, which was added to the game on the 4th October 2024.


Patterns

Like any other charm in the game, the Backsplash charm is pattern based. The pattern determines the look of the charm itself. For the Backsplash charm it changes the color of the charm, depending on the pattern it can be green, blue and purple.


Green Patterns

The patterns around pattern 20.000 on the charm will make the charm green. As the pattern gets closer to 20.000 it will be more green. This means pattern 20.000 is the best pattern and the pattern where the charm is as green as it can get.

Blue Patterns

The patterns around pattern 80.000 on the charm will make the charm blue. As the pattern gets closer to 80.000 it will be more blue. This means pattern 80.000 is the best pattern and the pattern where the charm is as blue as it can get.

Purple Patterns

The patterns around pattern 100.000 on the charm will make the charm purple. As the pattern gets closer to 100.000 it will be more purple. This means pattern 100.000 is the best pattern and the pattern where the charm is as purple as it can get.
