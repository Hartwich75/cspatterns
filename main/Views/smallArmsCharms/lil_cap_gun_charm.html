Lil' Cap Gun Charm Pattern Guide



Lil' Cap Gun Charm

The Lil' Cap Gun Charm is a high grade collectible from the Small Arms Charm collection, which was added to the game on the 4th October 2024.


Patterns

Like any other charm in the game, the Lil' Cap Gun charm is pattern based. The pattern determines the look of the charm itself. For the Lil' Cap Gun charm it changes the color of the gun tip, depending on the pattern it can be green, yellow and orange.


Green Patterns

The low patterns on the charm will make the gun tip green. As the pattern gets closer to 1 it will be more green. This means pattern 10 is the best pattern and the pattern where the gun tip is as green as it can get.

Yellow Patterns

The mid patterns on the charm will make the gun tip yellow. As the pattern gets closer to 50.000 it will be more yellow. This means pattern 50.000 is the best pattern and the pattern where the gun tip is as yellow as it can get.

Orange Patterns

The high patterns on the charm will make the gun tip orange. As the pattern gets closer to 100.000 it will be more orange. This means pattern 100.000 is the best pattern and the pattern where the gun tip is as orange as it can get.
